# Lazy Element
* 프로그래밍 적인 요소(변수, 메서드, 클래스 등)를 만드는 이유
  * 나중에 발생할 변화를 대비
  * 해당 함수 또는 클래스를 재사용
  * 의미있는 이름 부여 등
* 예상과 다르게 사용되지 않는 요소들이 생긴다 -> 이를 제거해야 한다
